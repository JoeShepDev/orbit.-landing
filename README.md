Landing page for Orbit. — a BLE-based proximity safety app for streamers and content creators.

🔗 Live: orbit-landing-fawn.vercel.app

Overview

Orbit. detects unauthorized Bluetooth devices entering a creator's physical space and triggers an escalating alert system. This landing page introduces the product, its core features, and its privacy-first design philosophy.

About Orbit.

Streamers and content creators invite the public into their world — but not always into their space. Orbit. runs in the background, scanning for unknown BLE devices on a configurable interval, and escalates through a three-strike alert system when something unexpected shows up.

Core features:


Interval-based BLE scanning
Three-strike escalating alert system
Radar-style proximity UI
Hashed MAC address storage — no raw device identifiers stored
30-day auto-delete on all stored data
High-noise mode for busy environments


Tech Stack


Framework: React + Vite
3D / Animation: Three.js + React Three Fiber
Backend / Auth: Supabase
